Hello! Welcome to my README!

To run the program please build the .java file by navigating to the directory L4.java is stored in
using a terminal and run this command "javac L4.java"
After that you can run the program by using the command "java L4"

The outputs you get for L4.java are in L4_Output.txt and for L4_Queries.sql the output is in L4_Queries_Output.txt
